FreeCodeCamp API Basejump: Image Search Abstraction Layer

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example search usage:

https://brendobrien-imagesearch.herokuapp.com/search/corn

Example creation output

{ "image_URL":"http://i.imgur.com/2Ig2nIr.jpg", "alt_text":"The story of Shay (almost) eating a corn fritter.", "page_url":"http://imgur.com/user/Yoshiboy100/submitted" }

To paginate search:

https://brendobrien-imagesearch.herokuapp.com/search/corn?offset=10

List of searches

https://brendobrien-imagesearch.herokuapp.com/list
